Política de Privacidad

Última actualización: 2025-11-13

Esta aplicación (“MAGIARS”) procesa información únicamente con el propósito de permitir la interacción con las APIs de Meta (Facebook/Instagram) para fines de prueba, desarrollo y funcionamiento básico del servicio conectado.

Recopilación de información

La Aplicación puede recibir datos proporcionados automáticamente por las plataformas de Meta, incluyendo:

  • Identificadores de usuario de Instagram o Facebook
  • Mensajes enviados a cuentas conectadas
  • Información necesaria para la autenticación mediante API

Uso de la información

La información se utiliza exclusivamente para:

  • Procesar y responder mensajes
  • Probar el funcionamiento del sistema
  • Gestionar integraciones con las APIs de Meta

La información no se comparte, vende ni distribuye a terceros.

Retención de datos

Los datos se mantienen únicamente el tiempo necesario para el funcionamiento básico o pruebas. Pueden ser eliminados a solicitud del usuario o administrador.

Seguridad

Se aplican medidas razonables para proteger la información contra acceso no autorizado. Esta Aplicación puede ejecutarse en entornos de prueba.

Servicios de terceros

La Aplicación interactúa únicamente con:

  • Meta (Facebook / Instagram Graph API)
